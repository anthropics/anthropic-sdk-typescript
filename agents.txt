## Summary

The official TypeScript SDK for the Anthropic API. Provides programmatic access to Claude models from server-side TypeScript or JavaScript applications running on Node.js 18+.

## Capabilities

- Read the full source tree (most of it is auto-generated from Anthropic's OpenAPI spec via Stainless).
- Modify files under `src/lib/**` — hand-written, generator-safe utilities (streams, parsers, helpers).
- Modify files under `examples/**` — standalone example scripts, also generator-safe.
- Run tests with `yarn run test`.
- Run linters with `yarn lint`.
- Create example scripts demonstrating API usage.
- Update documentation comments and JSDoc.
- Suggest or implement bug fixes in hand-written code paths.

## Prohibitions

- Do NOT publish to npm (`publish_to_npm` is prohibited). Releases go through the automated GitHub Actions pipeline.
- Do NOT merge to `main` (`merge_to_main` is prohibited). All changes require human review and approval.
- Do NOT modify secrets, tokens, or credentials (`modify_secrets` is prohibited).
- Do NOT modify generated code outside `src/lib/**` and `examples/**`. These paths are owned by the Stainless generator and will be overwritten on regeneration.
- Do NOT open issues or PRs on behalf of the repository maintainers without explicit user instruction.

## Entry Points

- test_command: yarn run test
- lint_command: yarn lint
- build_command: yarn build

## Generator-Owned Paths

Most of the SDK source is auto-generated by [Stainless Software Inc](http://stainless.com) from Anthropic's OpenAPI specification. The generator will never modify `src/lib/**` or `examples/**`, which are explicitly carved out as safe for hand edits per CONTRIBUTING.md.

All other paths require human approval for modification.
